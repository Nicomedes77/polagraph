## Materiales
- Tabla de madera (1)
- Arduino MEGA 2560 (1)
- RAMPS 1.4 (1)
- Motor NEMA 17 (2)
- Switch final de carrera (2)
- Bulón 3 1/2 (4)
- Correa GT2 6mm (3m)
- Polea dentada 60 dientes para correa GT2 eje 5mm (2)
- Piezas impresas en 3D para el cabezal
- Módulo REPRAP Smart Controller
